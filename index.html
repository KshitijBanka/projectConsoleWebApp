Project on Console

Visual Studio is a great coding platform that helps a lot. You can click on Open folder and then upload a folder where you can create files(example 'index.html') and then code.

You can print different things on the console for testing purposes. To open it, you have to right click, click on inspect and then press Console. You'll then be able to see the output. To write a code to print something on the console, you have to type: console.log() and enter anything inside. For strings, put it in quotes.
